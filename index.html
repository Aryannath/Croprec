ABOUT THE PROJECT

This is a machine learning state-of-the-art model with an accuracy of over 99%

In this model we have used a XGBoost Classifier which will predict for us the best crop out of 22 labels for crops available, which can be planted on the field based on diffrent factors of the soil like nitrogen content of the soil and rainfall levels. For getting the temprature of the next 15 days and the humidity of the area we are using OpenWeather's free API for developers. The machine learning model is hosted on this interactive website using Heroku. The dataset used for training our model was taken up from Kaggle which had 2200 instances and 7 variables.
